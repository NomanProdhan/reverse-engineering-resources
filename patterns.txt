## ragg2 examples for generating patterns

# Generate pattern of 50 characters for 32 bit system
ragg2 -b 32 -P 50 -r

# Generate pattern of 50 characters for 64 bit system
ragg2 -b 64 -P 50 -r

# Add a string at the end of pattern
ragg2 -b 64 -P 50 -r -S customStringOrPayloadHere
